一、网页三层结构
1.结构层 html ==> 网页内容控制
2.表示层 css ==>  网页的布局和美化
3.行为层 JavaScript ==> 网页的智能(行为)部分(网页功能的实现，验证，特效，控制...)

二、JS==>ES6
1.内嵌式语言(js/php)==>脚本
2.变量(varisble):名称不变，值根据需要不断改变的量
time，name
3.变量的命名规则:
a.必须是字母或_开头，不能用数字或者特殊符号。
b.当变量中出现两个单词时，第二个单词开始首字母大写。(驼峰命名)
4.变量赋值的方式:
a.直接赋值: var name="Tom";
b.间接赋值:一个变量的值来自另外一个变量
