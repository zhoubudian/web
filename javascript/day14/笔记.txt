一.循环语句: 在循环次数内，周而复始地执行一段代码
注意:循环条件(小括号中的内容)必须在某一时刻为false,才能是循环停止.
1.for(循环条件){
	代码块
}
用途:往往用于循环次数固定或已知循环次数的场合.
2.while:
while(条件){
	代码块
}
用途:
往往用在未知循环次数的情况
3.do while:
do{
	代码块
}while();
注:即使循环条件为假,也会至少执行一次.

二.死循环
当循环条件永远为真时,将会出现无休止的死循环!
记住:代码中一定避免死循环出现.
while(1){
	alert("aa");
}
